Installation

Voici les notes prises pendant l’installion.

télécharge l’installateur

  1. Aller sur le site de systema
  2. Créer un compte, attendez l’activation, logger vous
  3. cliquer “Download”
  4. dans le menu de gauche : Dowload > Thermica > V4 > Thermica X.Y.Z
  5. Télécharger la dernière version, aujourd’hui c’est la 4.9.0 L’installateur s’appel Systema-Thermica-Thermisol-Libsolver-4.9.0-64LNX pour linux, et fait env. 500MB

Installer sous Linux

J’utiliser aujorud’hui Fedora 33.

  1. Déplacer l’installateur téléchargé dans votre dossier d’installation.
  2. ajouter le droit d'écriture à l’installateur chmod +x Systema-Thermica-Thermisol-Libsolver-4.9.0-64LNX
  3. Lancer l’installateur comme un executable
  4. cliquer next
  5. accpeter la licence
  6. Choisissez le dossier d’installation
  7. choisissez tout à installer
  8. attendez

Lancer Thermica

Depuis le dossier d’installation : ./SystemaLNX

Il faut ajouter le ‘Home’ à systema : c’est le dossier d’installation : SYSTEMA_HOME=/home/tavant/Data/tavant/CSE-perso/Thermica/themrcia_install_4.9/Systema-4.9.0/ ./SystemaLNX

Problem librairy libstdc++

Dans le cas d’un message d’erreur :

./SystemaLNX: /home/tavant/Data/tavant/CSE-perso/Thermica/themrcia_install_4.9/Systema-4.9.0/lib/libstdc++.so.6: version `CXXABI_1.3.9' not found (required by /lib64/libGLU.so.1)

C’est parce que la librairy libstdc++ shipé avec l’installateur est trop vielle (2016 dans mon cas). Il y a deux solutions :

  • quick : ajouter LD_PRELOAD="\lib64/libstdc++.so.6" avant de lancer ./SystemaLNX
  • long : modifier la librairy shipé par la nouvelle.
